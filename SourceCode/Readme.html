💎 COMO INSTALAR

1. Descomprima o pacote .7z
2. Abra o PowerShell
3. Habilite a execução de scripts para o usuário atual
Set-ExecutionPolicy Unrestricted -Scope CurrentUser -Force
4. Desbloqueie o script
Get-Item Autopilot.ps1 | Unblock-File
5. Execute o script
.\Autopilot.ps1

Certifique-se de informar o local correto do script.
